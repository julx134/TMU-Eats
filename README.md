# CPS714-Project

To start the project:
1) You must have node.js installed at https://nodejs.org/en/
2) Check that node.js was installed properly by running: <br />
  node -v <br />
  npm -v <br />
3) cd into tmu-eats/ 
4) type npm start
Append "/home" for homepage dev
Append "/cart" for cart page dev
